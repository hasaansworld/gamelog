# Gamelog

Gamelog is an online library manager for all your games.
All data is saved locally in the browser.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

```

## Development Server

Start the development server on http://localhost:1337

```bash
npm run dev
```
