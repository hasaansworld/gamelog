# Backlog.rip

Based on a Nuxt 3 project.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

```

## Development Server

Start the development server on http://localhost:1337

```bash
npm run dev
```

### Emoji Cheat Sheet

💽 - Used in $db or dexie
